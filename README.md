# raw-converter

## Usage

```
Raw Photo Converter v0.0.1
Based on LibRaw, version: 0.19.5-Release
Usage:
  rc [OPTIONS] raw-file > target.ppm
  rc [OPTIONS] raw-file | cjpeg -quality 80 > target.jpg
Options:
  -L      list supported cameras and exit
  -v      Print verbose messages
  -T      output TIFF files instead of .pgm/ppm
  -o      Specity output file
  -w      Specify output width
  -h      Specify output height
```
